# Chair Bracket

A 3D-printable chair bracket modeled in FreeCAD. The geometry is a lofted
solid ("Chair Bracket-Loft"), exported for printing as both STL and 3MF.

## Files
- `Chair Bracket.FCStd` / `Chair Bracket.FCStd1` — FreeCAD documents (binary; `.FCStd1` is the prior-save backup).
- `Chair Bracket-Loft.stl`, `Chair Bracket-Loft.3mf` — printable output of the lofted bracket body.

## Usage
Open `Chair Bracket.FCStd` in FreeCAD 1.0.2 to view or edit the model. The
printable mesh is provided in `Chair Bracket-Loft.stl` / `.3mf`. No build
scripts are present in this directory.
