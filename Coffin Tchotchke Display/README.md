# Coffin Tchotchke Display

A 3D-printable coffin-shaped display piece with shelves, for showing off small
trinkets (tchotchkes). Modeled in FreeCAD and exported for printing.

## Files
- `Coffin v1.FCStd` / `Coffin v1.FCStd1` — FreeCAD documents (binary; `.FCStd1` is the prior-save backup).
- `Coffin v1-Body.stl` — printable coffin body mesh.
- `Coffin v1-w:Shelves.3mf` — printable coffin including shelves.

## Usage
Open `Coffin v1.FCStd` in FreeCAD 1.0.2 to view or edit the model. Printable
output is provided as `Coffin v1-Body.stl` and `Coffin v1-w:Shelves.3mf`. No
build scripts are present in this directory.
