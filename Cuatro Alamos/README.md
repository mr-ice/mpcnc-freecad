# Cuatro Alamos

A FreeCAD model of a main cabin ("Main Cabin v1"). The directory currently
contains only the FreeCAD document and its backup; no exported print meshes or
build scripts are present, so the exact intent (e.g. architectural model vs.
printable part) is not fully evident from the files.

## Files
- `Main Cabin v1.FCStd` / `Main Cabin v1.FCStd1` — FreeCAD documents (binary; `.FCStd1` is the prior-save backup).

## Usage
Open `Main Cabin v1.FCStd` in FreeCAD 1.0.2 to view or edit the model.
