# Dive Lights

FreeCAD models for printed inserts related to dive lights, including a
standard insert and a UV variant. The project produces 3D-printable STL and
3MF meshes sized roughly 171x99 mm and 190x100 mm.

## Files
- `Insert.FCStd` / `Insert.FCStd1` — FreeCAD document (and backup) for the standard insert
- `UVInsert.FCStd` — FreeCAD document for the UV insert
- Output meshes:
  - `Insert 171x99.stl`, `Insert 171x99 fillets.stl`, `Insert 171x99 fillets.3mf` — 171x99 mm insert (plain and filleted)
  - `UVInsert 190x100 fillets.stl` — 190x100 mm filleted UV insert

## Usage
Open the `.FCStd` documents directly in FreeCAD 1.0.2 to edit, then export the
meshes. No Python script, macro, or config is present in this directory.
