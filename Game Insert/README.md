# Game Insert

A collection of 3D-printable board-game box inserts and organizers, each modeled in FreeCAD. Every sub-project is self-contained and has its own README (or project notes) describing its parts and usage.

## Sub-projects
- `Robo Rally 30th Anniversary` — Insert for the Robo Rally 30th Anniversary edition: big-box tray, card-deck holders, map/board storage, player kits, and misc parts.
- `Terraforming Mars` — Card-wedge holders for the Terraforming Mars box.
- `Wispwood` — Insert project with its own `PROJECT.md` and config (see that folder).

## Usage
Open the desired sub-project, then open its FreeCAD document or macro in FreeCAD 1.0.2 and run/recompute it. See each sub-project's README/notes for details and exported STL/3MF outputs.
