# Terraforming Mars Insert

3D-printable card-wedge holders for the Terraforming Mars board-game box. The wedges sit under stacks of cards to angle them up for easier access/storage.

## Files
- `Card Wedge.FCStd` — FreeCAD document for the card wedge.
- `Card Wedge.stl` — Default exported wedge mesh.
- `Card Wedge 105x25.stl` / `Card Wedge 105x25.3mf` — Wedge sized 105 x 25 mm.
- `Card Wedge 110x20.stl` — Wedge sized 110 x 20 mm.

## Usage
Open `Card Wedge.FCStd` in FreeCAD 1.0.2, adjust dimensions as needed, and export to STL/3MF for printing. Pre-exported variants are included for common card-stack sizes.
