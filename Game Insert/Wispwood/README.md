# Wispwood Tile Holder

A custom two-stack tile rack for the game Wispwood, with a two-part sliding/dispensing lid and an integrated folding stand. See [PROJECT.md](PROJECT.md) for the full design spec and parameters.

## Files
- `Wispwood.FCMacro` — FreeCAD macro: reloads `config` and `wispwood`, rebuilds the `Wispwood` document from `wispwood.build_all()`.
- `wispwood.py` — parametric geometry (tray, lid, scoops, folding stand).
- `config.py` — design constants.
- `gepesso/` — reference STLs from the community wispwood-tile-holder this design is derived from.
- `Wispwood-*.stl` — exported parts (Tray, Stand, TrayStand, LidLarge, LidSmall).
- `wispwood-tile-holder20251205-1-dqw5ug.zip` — archived reference download.
