# Grill Magnet Holder

FreeCAD models for a magnet holder for a grill, iterated across three versions.
The project produces a 3D-printable STL of the latest (v3) holder.

## Files
- `Magnet Holder v1.FCStd` — first version
- `Magnet Holder v2.FCStd` / `Magnet Holder v2.FCStd1` — second version (and backup)
- `Magnet Holder v3.FCStd` / `Magnet Holder v3.FCStd1` — third version (and backup)
- `Magnet Holder v3.stl`, `Magnet Holder.stl` — exported holder meshes (identical files)

## Usage
Open the `.FCStd` documents directly in FreeCAD 1.0.2 to edit, then export the
mesh. No Python script, macro, or config is present in this directory.
