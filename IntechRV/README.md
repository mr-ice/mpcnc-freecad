# Intech RV Cabinet Plug

A FreeCAD model of a cabinet plug for an Intech RV, exported at several
diameters. The project produces 3D-printable STL plugs in 25, 26, and 27 mm
sizes for a press fit.

## Files
- `CabinetPlug.FCStd` — FreeCAD document for the cabinet plug
- Output meshes:
  - `CabinetPlugv1-25mm.stl` — 25 mm plug
  - `CabinetPlugv1-26mm.stl` — 26 mm plug
  - `CabinetPlugv1-27mm.stl` — 27 mm plug

## Usage
Open `CabinetPlug.FCStd` directly in FreeCAD 1.0.2 to edit, then export the
desired-diameter mesh. No Python script, macro, or config is present in this
directory.
