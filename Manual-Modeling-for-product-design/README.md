# Manual Modeling for Product Design

Learning and reference material for hand-driven (GUI) FreeCAD modeling rather
than a finished printable project. These are tutorial/practice documents, not
script-generated parts.

## Files
- `Basic Part Design Tutorial.FCStd` — FreeCAD document following a basic Part Design tutorial.
- `lego-test.FCStd` — FreeCAD practice document modeling a LEGO-style brick.
- `lego-test.FCStd1` — backup of the previous save of `lego-test.FCStd`.

## Usage
Open the `.FCStd` files in FreeCAD 1.0.2 to inspect or continue the modeling
exercises. No build macro or output mesh is included; these are reference
documents.
