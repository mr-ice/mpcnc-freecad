# MaterMaze

A collection of 3D-printable tabletop terrain projects modeled in FreeCAD. Each sub-project is self-contained and has its own README.

## Sub-projects
- `Water` — Procedurally generated water terrain tiles (rippled NURBS surface) in 1x1 through 3x3 grid sizes, with channel-cut square edges.

## Usage
Open the desired sub-project and run its FreeCAD macro/script in FreeCAD 1.0.2. See each sub-project's README for details and exported STL/3MF outputs.
