# MicroSwitch

A collection of microswitch / endstop housing meshes for 3D printing. Holds
exported STL models only (no source script or FreeCAD document in this folder).

## Files
- `EndStop-Microswitch-Housing2.stl` — endstop microswitch housing, version 2.
- `Microswitch-Holder-v3.stl` — microswitch holder, version 3.
- `MicroSwitchHousing-frompython.stl` — microswitch housing exported from a Python-driven model.

## Usage
Load any `.stl` into a slicer and print. These are output meshes; the generating
sources are not stored in this directory.
