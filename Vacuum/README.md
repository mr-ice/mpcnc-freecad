# Vacuum Hose Adapters

3D-printable vacuum hose reducer adapters modeled in FreeCAD 1.0.2. Two variants
step a 2.5" port down to a smaller diameter (1.4" and 1.5"), exported as STL/3MF
for printing.

## Files
- `2.5 to 1.4.FCStd` / `.FCStd1` — FreeCAD model (and backup) of the 2.5"-to-1.4" reducer.
- `2.5 to 1.4-Fusion001.stl` — printable export of the 2.5"-to-1.4" adapter.
- `2.5 to 1.5.FCStd` / `.FCStd1` — FreeCAD model (and backup) of the 2.5"-to-1.5" reducer.
- `2.5 to 1.5-Fusion001.stl` / `.3mf` — printable exports of the 2.5"-to-1.5" adapter.

## Usage
Open the `.FCStd` files in FreeCAD 1.0.2 to edit. Print the `*-Fusion001.stl` /
`.3mf` exports. No build scripts or macros are present.
