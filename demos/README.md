# Demos

A folder of demo/sample FreeCAD files used for reference and experimentation.
It is not a parametric project of its own — there is no Python script, macro,
or config here.

## Files
- `radiaatori_kandur_normandc1.fcstd` — sample/demo FreeCAD document (binary)

## Usage
Open the `.fcstd` file directly in FreeCAD 1.0.2 to inspect it.
